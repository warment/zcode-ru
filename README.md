# 🇷🇺 ZCode — Русская локализация

**Замена китайского языка на русский в [ZCode 3.0.1](https://zcode.z.ai)**

Скрипт патчит оригинальное приложение: китайский (`zh-CN`) заменяется на русский. В итоге в настройках остаются два языка — **English** и **Русский**.

## Зачем это нужно

ZCode поддерживает только **китайский** и **английский**. Русского нет, и добавить его через интерфейс невозможно — локали зашиты в JS-бандл.

Этот скрипт решает проблему за одну команду:

- ✅ **Патч оригинального приложения** — без копий и дублей
- ✅ **Китайский заменяется русским** — в меню остаются English и Русский
- ✅ **3 528 ключей переведено** (97% от оригинального словаря)
- ✅ **Работает после обновлений** — повтори команду, и всё снова на русском

## Быстрый старт

```bash
git clone https://github.com/warment/zcode-ru.git
cd zcode-ru
./tools/patch.sh
```

**Важно:** закрой ZCode перед запуском скрипта.

Скрипт автоматически:
1. Распакует `app.asar` из **оригинального** `/Applications/ZCode.app`
2. Заменит китайский словарь на русский
3. Пропатчит валидацию и меню выбора языка
4. Перепакует `app.asar` и переподпишет приложение

После этого открой ZCode — в настройках будут **English** и **Русский**.

## Как работает

### Подход

```
Оригинал ZCode.app          После патча
┌──────────────────┐        ┌──────────────────┐
│  zh-CN (Китайский)│   →   │  ru (Русский)     │
│  en-US (English)  │       │  en-US (English)  │
└──────────────────┘        └──────────────────┘
```

Китайский словарь (`sv`) в JS-бандле **полностью заменяется** русским (`ru.json`). Английский остаётся без изменений. В результате пользователь видит два языка в настройках: **English** и **Русский**.

### Замена словаря

В файле `usageStatsUiParts-*.js` находится объект `sv` — это китайский словарь (~3 000 ключей). Скрипт находит его границы по фигурным скобкам и заменяет содержимое на `ru.json`.

Размер: 189 KB → 204 KB (+15 KB). Парсер не ломается.

### Патчи кода

После замены словаря нужно обновить точки, где код проверяет допустимые локали:

| Файл | Что патчится |
|:-----|:-------------|
| `index-*.js` | Добавление `"ru"` в объект `nNt` |
| `index-*.js` | Диспетчер перевода `JQ()` |
| `index-*.js` | Валидация `Zat()`, `wut()` |
| `index-*.js` | Маппинг локаль→страна `Xat()` |
| `index-*.js` | Mobile handler, settings handler |
| `index-*.js` | Ключи localStorage |
| `chunk-*.js` | Zod-валидация, fallback по `navigator.language` |

## Пересборка после обновления

ZCode обновляется автоматически — перевод слетает. Пересобери за одну команду:

```bash
cd zcode-ru && ./tools/patch.sh
```

Или укажи свой путь:

```bash
./tools/patch.sh /path/to/ZCode.app
```
