# Welcome to the New ZCode

ZCode is a full-featured Agentic Development Environment (ADE) built for **Long Horizon Tasks**. It helps AI Agents complete long-running, multi-step development work end-to-end with stability and control.

ZCode focuses on two core strengths: **long-running tasks** and **stability**. The self-developed ZCode Agent keeps goals, files, terminal results, browser context, execution modes, and Git state inside the same task so complex development work can move from planning to verification without losing continuity.

With natural-language instructions, you can ask ZCode Agent to handle coding, debugging, testing, project preview, and change review. Through the desktop workspace, Remote, and Bot Channel, you can keep checking progress and adding instructions while long-running work continues.

## Core Capabilities

ZCode organizes the development experience around the self-developed ZCode Agent, with emphasis on:

- **Long-task execution**: keep requirement understanding, planning, code changes, verification, and Review inside the same task.
- **Stable context retention**: combine files, terminal output, browser context, execution modes, and Git state to reduce mid-task context breaks.
- **Self-developed Agent workflow**: tasks, permissions, context, tool calls, and review flow are organized around ZCode Agent.
- **Continuous follow-up across devices**: desktop, mobile Remote, and Feishu / WeChat Bot can all keep the same workspace task moving.
- **Safe and controllable execution**: sensitive commands, file changes, and high-permission actions go through confirmation before execution.

* * *

## Release Highlights

### In-house ZCode Agent

Purpose-built optimizations for Long Horizon Tasks, with significantly better stability and context retention across long-running workflows.

### Unified Agent workflow

ZCode Agent unifies tasks, context, permissions, file references, and Review so long-running work stays continuous.

### Feishu / WeChat Bot integration

Bot capability is now available in Feishu and WeChat — just @ the bot to let ZCode Agent trigger and advance tasks remotely.

### Mobile Remote control

Remote capability now supports controlling an Agent from your phone with parity to the desktop experience, enabling Vibeworking.
