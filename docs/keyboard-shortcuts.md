# Keyboard Shortcuts

This page keeps only the shortcuts that are most useful and worth remembering in ZCode. We focus on task flow, chat input, and daily high-frequency actions instead of listing every low-frequency item from the system menu.

* * *

## Global Actions

| Windows Key | Mac Key | Description |
| --- | --- | --- |
| `Ctrl` +`N` | `Command` +`N` | Create a new task |
| `Ctrl` +`B` | `Command` +`B` | Toggle sidebar |
| `Ctrl` +`J` | `Command` +`J` | Toggle terminal panel |
| `Ctrl` +`Alt` +`B` | `Option` +`Command` +`B` | Toggle the right side pane |
| `Ctrl` +`K` | `Command` +`K` | Open the command center |
| `Ctrl` +`F` | `Command` +`F` | Find in conversation |
| `Ctrl` +`[` | `Command` +`[` | Go back to the previous task or workspace |\
| `Ctrl` +`]` | `Command` +`]` | Go forward to the next task or workspace |
| `Ctrl` +`Shift` +`[` | `Shift` +`Command` +`[` | Switch to the previous conversation |\
| `Ctrl` +`Shift` +`]` | `Shift` +`Command` +`]` | Switch to the next conversation |

* * *

## Quick Pickers

| Windows Key | Mac Key | Description |
| --- | --- | --- |
| `Ctrl` +`M` | `Control` +`M` | Open the model picker |
| `Shift` +`Tab` | `Shift` +`Tab` | Cycle execution modes (while the chat input is focused) |
| `Ctrl` +`T` | `Control` +`T` | Open the Thought Level picker |
| `↑ / ↓` | `↑ / ↓` | Move through options in Slash or Mention panels |
| `Enter / Tab` | `Enter / Tab` | Select the highlighted option |
| `Esc` | `Esc` | Close the current panel or menu |

* * *

## Panels And Diff View

The Git diff panel has no dedicated shortcut: run "Toggle diff panel" from the command center (Command + K), or open the **Review** tab in the right side pane to inspect changes.
