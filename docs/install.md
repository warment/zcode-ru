# Install

This section shows how to download and install the new ZCode desktop app.

* * *

## Download

[Download ZCode\\
\\
Download .dmg(Apple Silicon)](https://cdn.zcode-ai.com/zcode/electron/releases/3.0.1/ZCode-3.0.1-mac-arm64.dmg) v3.0.1·Latest

All platforms

[ZCODE for macOS (Intel)](https://cdn.zcode-ai.com/zcode/electron/releases/3.0.1/ZCode-3.0.1-mac-x64.dmg) [ZCODE for Windows](https://cdn.zcode-ai.com/zcode/electron/releases/3.0.1/ZCode-3.0.1-win-x64.exe)

Currently supported platforms include macOS (Apple Silicon / Intel), Windows, and mainstream Linux distributions.

* * *

## Quick Installation Guide

### Windows

Download the installer, double-click it, and follow the setup wizard to complete installation.

### macOS

Open the downloaded DMG file, drag **ZCode.app** into the **Applications** folder, and then launch it from Launchpad.

### Linux

Download the package for your distribution and install it using the standard installation flow for that system.

* * *

## Verify Installation

Open the app and complete sign-in to begin using the new AI-assisted development workflow.
